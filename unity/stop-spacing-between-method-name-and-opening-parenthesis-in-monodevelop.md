# Stop spacing between method name and opening parenthesis in MonoDevelop

Select the menu of MonoDevelop-Unity:

**Preferences** > **Source Code** > **Code Formatting** > **C# source code**

Then select **C# Format** and click **Edit** button.

In *Edit Profile* diplayed, set *Category* to **White Space** at first.

---

To stop spacing between method name and opening parenthesis when methods are declared:

Uncheck the box of **Declarations** > **before opening parenthesis**.


To stop spacing when methods are called as well:

Uncheck the box of **Expressions** > **Method invocations** > **before opening parenthesis**.
